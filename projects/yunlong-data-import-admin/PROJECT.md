# PROJECT

## 项目基本信息
- 项目名称：yunlong-data-import-admin
- 端类型：admin
- 对应规则集：admin-ui
- 默认演示视口：1366x768；兼顾 1440x900
- 项目用途：演示学校数据表格批量导入平台的后台管理端原型
- 目标用户：检测机构/教育局侧运营人员、项目团队成员、演示评审人员

## 当前阶段
- 当前阶段：05-高保真原型实现

## 项目范围
本项目用于把检测机构交付的多学校学生数据表格导入后台系统，以 mock 方式演示批量导入、状态进度、异常导出和导入历史。

## 关键约束
- 只做后台管理端，不做移动端。
- 默认本地 mock，不接真实后端。
- 不真实解析 Excel，仅模拟 xlsx/xls/csv 文件选择与导入结果。
- 不在 05 阶段新增 02 未定义的页面、字段、按钮或模块。
- 后续必须按 01 -> 02 -> 03 -> 04 -> 05 顺序推进。
