### 9.4 交付检查（含低级错误防线）
- 本项目额外禁止事项：
- 交付前检查项：
  - 主色是否统一（按钮/图表主线/重点标签一致）：
  - 页面背景与卡片层级是否可辨：
  - 纯白背景或无阴影卡片是否有边界支撑：
  - 移动端底部固定区是否遮挡末项：
  - 触控热区或桌面点击区是否达标：
  - 后台控件尺寸是否符合 14px/32px 默认口径：
  - 表格进度条是否为 6-8px 细条：
  - 卡片 Header 是否默认无横线：
  - 图标库、尺寸、线宽是否统一：
  - 图片来源、用途和敏感场景限制是否明确：
  - 空态/成功/异常态是否为“图标+文案+下一步动作”：
  - 图表主色是否使用项目 token（非默认色）：
  - 图表是否具备 Tooltip、Legend、空态/加载态：
  - 后台表格密度、列宽、按钮优先级是否清晰：
  - 后台卡片间距是否统一：
  - 后台页面是否避免无控制整体滚动：
  - 是否避免明显组件库默认感：
