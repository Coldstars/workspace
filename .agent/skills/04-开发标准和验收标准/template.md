## 9. 验收清单
- [ ] app 可安装依赖并启动
- [ ] 项目端类型与规则集匹配

- [ ] 主演示链路可一遍走通
- [ ] 页面返回路径清晰
- [ ] mock 数据无明显穿帮
- [ ] 验收视口下无横向滚动、无底部遮挡
- [ ] 移动端主按钮和图标按钮触控区域符合默认口径
- [ ] 后台 AppShell/Sidebar/Topbar/面包屑符合 02/03（admin 适用）
- [ ] 后台表格列、筛选、批量操作、行级操作与 02 一致（admin 适用）
- [ ] 后台导出、重试、异常处理可演示（admin 适用）
- [ ] 后台表格或内容区内部滚动策略正确（admin 适用）
- [ ] 后台按钮、输入框、选择器默认尺寸符合 14px/32px（admin 适用）
- [ ] 后台表格进度条为 6-8px 细条（admin 适用）
- [ ] 后台卡片 Header 默认无横线，除非 03/04 明确例外（admin 适用）
- [ ] Tab、筛选、表单、提交、弹窗、图表交互可用
- [ ] 色彩、字号、圆角、阴影、间距一致
- [ ] Tailwind 完成主要样式，避免明显组件库默认视觉
- [ ] 空态、成功态、告警态有图标辅助
- [ ] 新增依赖均在允许依赖清单中声明
- [ ] 图标库一致，图标按钮具备可访问名称
- [ ] 图表 tooltip、legend、空态、加载态可演示
- [ ] 图片来源、授权边界和业务适配已确认
- [ ] 图片、图表和异步内容加载后无明显布局跳动
- [ ] 可点击元素具备 press/active/focus-visible
- [ ] 图标按钮具备可访问名称，表单项具备可见 label
- [ ] 状态信息不只依赖颜色
- [ ] 动效尊重 prefers-reduced-motion
