# 附录：配色模板库

## 使用方式
- 仅在 03 视觉规范需要选择或校准配色时读取。
- 每套模板都必须映射为语义 token：`primary / on-primary / secondary / accent / background / foreground / card / muted / border / destructive / ring`。
- 允许按品牌替换主色，但必须保留语义 token 结构和状态色含义。
- 单页面不得随机散落 hex。
