# 后台表格与筛选规范

## 表格规格必须定义
- 数据对象名称。
- 列名、字段含义、展示格式。
- 主键或可识别字段。
- 状态列与状态枚举。
- 进度列、数量列、时间列等格式。
- 行级操作：查看、编辑、删除、导出、重试等。
- 批量操作：批量开始、批量导出、批量删除、批量标记等。
- 空态、加载态、异常态、筛选无结果。

## 筛选规格必须定义
- 搜索字段。
- 状态筛选。
- 时间筛选。
- 组织/分类筛选。
- 默认筛选值。
- 重置筛选行为。

## 表格体验
- 表头文案短而明确。
- 状态用标签或图标+文案，不能只靠颜色。
- 操作列固定在右侧时需考虑窄屏可用性。
- 数据多时优先表格内部滚动；页面整体不应因为表格变成长滚动。
- 分页、虚拟滚动或内部滚动策略必须在 02/04 说明。

## 验收
- 表格列与 02 完全一致。
- 行级操作有真实反馈。
- 批量操作有进度或结果摘要。
- 导出按钮在无可导出数据时有禁用或提示。
